Cybersecurity Detection & Monitoring Lab

Enterprise Network Simulation & Threat Detection

Project Overview

Designed and implemented a comprehensive virtualized homelab environment that simulates an enterprise network infrastructure. This lab provides a safe, controlled environment for security testing, threat detection, and incident response practice.

pfSense Security Onion Splunk Kali Linux Windows Server Active Directory VMware

Infrastructure Components

  • Host System: AMD Ryzen 5 3600X, 32GB RAM, 1TB NVMe SSD
  • VMware Workstation Pro as the hypervisor platform
  • pfSense firewall with advanced network segmentation
  • Security Onion for IDS and security monitoring
  • Dedicated analysis workstation for SOC simulation
  • Active Directory environment with Windows Server 2019

Implementation Details

Network Segmentation

Implemented sophisticated network segregation using pfSense:

  • Dedicated management interface for administrative access
  • Separate network segments for attack testing (Kali Linux)
  • Isolated victim network for Windows domain environment
  • SPAN port configuration for Security Onion monitoring
  • Custom firewall rules for controlled access between segments

Security Monitoring Configuration

  • Security Onion deployment with full packet capture and IDS capabilities
  • Splunk SIEM implementation with Universal Forwarders on Windows hosts
  • Custom Windows event log collection and correlation
  • Automated alert generation for suspicious activities

Active Directory Integration

  • Windows Server 2019 configured as domain controller
  • Certificate services implementation for PKI infrastructure
  • Multiple domain-joined Windows 10 workstations
  • Comprehensive group policy management
